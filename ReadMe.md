 🍕 Pizza Sales Analysis Dashboard


# About the Project

This project was created to analyze pizza sales data using SQL and Microsoft Excel. The main aim was to understand sales performance, customer ordering patterns, and product demand through data analysis and visualization.

I used SQL to perform various analyses on the dataset and then created an interactive dashboard in Excel to present the results in a simple and meaningful way.

---

# Tools Used

* SQL Server
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers

---

# What I Analyzed

* Total Revenue
* Average Order Value
* Total Orders
* Total Pizzas Sold
* Average Pizzas per Order
* Daily Order Trends
* Hourly Order Trends
* Sales by Pizza Category
* Sales by Pizza Size
* Top 5 Best Selling Pizzas
* Bottom 5 Selling Pizzas

---

# Dashboard Highlights

The dashboard provides a quick overview of business performance through different KPIs and charts. It helps in understanding:

* Which pizza categories generate the most revenue
* Customer ordering patterns throughout the day
* Best and worst performing pizzas
* Sales distribution by pizza size
* Overall sales trends
