# 🍕 Pizza Sales Analysis Dashboard

## Overview

This project focuses on analyzing pizza sales data using SQL and Microsoft Excel. The objective was to transform raw sales data into meaningful business insights through data analysis and interactive dashboard visualization.

The project includes KPI tracking, sales trend analysis, category-wise performance evaluation, and product performance analysis to support data-driven decision-making.

---

## Technologies Used

* SQL Server
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

---

## Key Performance Indicators (KPIs)

* Total Revenue
* Average Order Value
* Total Orders
* Total Pizzas Sold
* Average Pizzas per Order

---

## Analysis Performed

### Sales Trend Analysis

* Daily Order Trends
* Hourly Order Trends

### Revenue Analysis

* Revenue by Pizza Category
* Revenue by Pizza Size

### Product Performance Analysis

* Top 5 Best Selling Pizzas
* Bottom 5 Selling Pizzas

---

## Dashboard Features

* Interactive KPI Dashboard
* Sales Performance Tracking
* Category-wise Revenue Analysis
* Time-based Order Analysis
* Product Performance Evaluation
